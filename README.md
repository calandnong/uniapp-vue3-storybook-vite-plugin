# 支持storybook中渲染uniapp的标签/组件/api

插件使用看此处文档：[点击此处跳转](./packages//unplugin-uniapp-h5//README.md)

## 运行环境
```bash
node
```

## 安装依赖

根目录下运行下面代码：
```bash
pnpm i
```

## 运行演示示例
根目录执行下面代码：
```bash
pnpm preview
```

## 编译演示示例
```bash
pnpm preview:build
```
